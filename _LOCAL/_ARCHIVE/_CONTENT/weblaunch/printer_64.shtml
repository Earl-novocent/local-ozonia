Ozonia MP Series UV Systems
Dec 15, 2004, 06:19

Ozonia MP Series UV systems operate using the most sophisticated power supplies in the UV industry. The UV lamps are powered by energy efficient electronic ballasts, which have a turndown capacity from 100% to 40% optimizing energy consumption.  Each system comes standard with UV intensity monitoring and automatic wiping systems.  The MP series control systems standardize around an Allen-Bradley® PLC with a monochrome touch-screen for the operator interface.  The net result is an extremely flexible UV system able to handle a wide range of flows with fluctuating UV transmission (clarity).  Key features of Ozonia’s MP series UV systems include:

  • Medium pressure UV lamps
  • Variable output electronic ballasts
  • Local or remote operation
  • Automatic or manual control
  • PLC logic control with SCADA communication capability
  • Variable UV output based on a 4-20mA input in the automatic mode (flow pacing)
  • UV intensity measurement with automatic adjustment (constant dose)
  • Full data logging and historic data storage
  • Automatic wiping systems with adjustable wiping frequency
  • Polished 316L Stainless Steel reactors with 150# ANSI connections
